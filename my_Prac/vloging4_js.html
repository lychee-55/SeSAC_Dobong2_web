### JS의 참조 방식 JS의 참조방식으로는 2가지가 있습니다. 내장방식과 링크방식입니다. #### JS 내장방식 : ``` ``` #### JS 링크방식 : ``` ``` 두 방식 다 어디서나 사용가능하나, head태그안에 링크방식을 사용할 경우 script에 defer을 사용하는 것을 권장합니다. 문서의 읽기 순서는 읽히는 순서에 따라서 작동하게 됩니다! ### JS 표기법 JS에 권장되는 표기법은 아래와 같이 3개가 있습니다.
  • snake_case : HTML과 CSS에서 사용하는 표기법이지만 자바스크립트에서도 사용가능합니다.
  • camelCase : 띄어쓰기마다 대문자로 변경해 사용하는 것으로, 변수나 함수에는 camelCase 혹은 snake_case를 사용합니다.
  • PascalCase : 모든 앞자리를 대문자로 사용하는 것으로, 인터페이스나 클래스를 만들때 사용합니다.
#### 중요! Javascript는 0기반 번호매기기로 작동되기에 모든 순서를 읽을때 1이 아닌 0부터 읽기 시작한다는 것을 명심하자! ### Javascript의 데이터 종류(자료형) Javascript에서 배운 데이터 종류는 총 7가지가 있었습니다.
  • String : String은 문자열로 따옴표를 사용하는 문자형 데이터를 뜻한다.
  • Number : Number는 정수 및 소수점 숫자를 나타내는 숫자형 데이터를 뜻한다
  • Boolean : Booleantrue와 false 이 두가지 값만 가지는 참,거짓 데이터를 뜻한다.
  • Undefined : Undefined는 값이 할당되지 않은 상태를 표기하는 미할당 데이터입니다.
  • Null : Null은 의도적으로 비어있음을 뜻하는 데이터 입니다.
  • Object : Array는 여러 데이터(다른 데이터 타입도 가능)를 한 번에 꾸려주는 배열 데이터 입니다. 접근법은 []이며, 숫자 순서는 [0]부터 시작합니다.
  • Array : Object는 데이터 꾸러미로, 작성순서는 중요치 않습니다. 접근 방식은 {}이며, 값에 접근할 때는 key의 이름으로 접근합니다.

  • let : let은 변수를 선언할 때 사용하는 코드로, 재사용이 가능하고, 값의 재할당 또한 가능합니다.
``` let a = 1 let b = 2 console.log(a) // 1 console.log(b) // 2 a = 3 console.log(a) // 3 // 이렇게 재할당 가능 ```
  • const : const는 값의 재할당이 불가능합니다!
### JS의 연산자! JS의 연산자로는 기본, 비교, 논리 총 3가지 연산자를 배웠습니다.
  • 기본 연산자
    • % : 나머지 연산자, num % 2 == 1 이면 홀수, num % 2 == 0이면 짝수입니다.
    • ** : 거듭 제곱을 뜻합니다.
    • 증가, 감소 연산자 : Num++은 증가, Num--은 감소 연산자 입니다.
  • 비교 연산자
    • 일치 연산자 : 변수의 값과 자료형을 비교할 수 있는 연산자 입니다.
      • A == B : A와 B의 값이 같으면 true, 아니면 false
      • A === B : A와 B의 값과 자료형이 같으면 true, 아니면 false
        • ``` let a = 1 let b = "1" console.log(a == b) // 값이 같아 true console.log(a === b) // 값과 자료형중 자료형이 같지 않아 false ```
      • A != B : A와 B의 값이 다르면 true, 아니면 false
      • A !== B : A와 B의 값과 자료형이 다르면 true, 아니면 false
  • 논리 연산자
    • || : or로 해석함, 여러개 중 하나라도 true면 true, 모든 값이 false면 false
    • && : and로 해석함, 모든 값이 true면 true, 하나라도 false면 false
    • ! : true면 false, false면 true
### JS의 함수 함수란 특정 동작을 수행하는 코드의 집합이라고 한다. 함수릐 선언방식으로는 3가지가 있는데 아래 코드와 함께 배워보자! #### 함수 선언문 : ``` function sayName(){ console.log('이것은 함수 선언문 입니다.') } ``` #### 함수 표현식 : ``` let sayName = function(){ console.log('이것은 함수 표현식 입니다.') } ``` #### 화살표 함수 : ``` let sayName = (a,b)=>{ return a + b } // 혹은 let showError = () =>{ alert('error!') } ``` * 위 세가지 사용시 주의할 점은 호출방식 입니다! 함수 선언문은 어디서나 호출 가능하지만 함수 표현식은 코드에 도달해야 생성 및 사용가능합니다. 화살표 함수는 선언후에 호출이 가능하다는 것을 잊지 말자!
