HTML 태그 종류 2-2-1. 기본 태그 2-2-2. form 태그 2-2-3. table 태그 ## 🧐HTML이란? 웹사이트의 모습을 기술하기 위한 마크업언어. 프로그래밍 언어가 아니라 마크업 정보를 표현하는 마크업 언어로 문서의 내용 이외의 문서의 구조나 서식 같은 것을 포함한다. 그렇다면, HTML은 어떻게 만드는가? ### ■ HTML 구조📱: ``` Document ``` html문서를 작성할 때는 꼭 !DOCTYPE html로 시작하여 문서 형식을 html5로 지정해야합니다.
## 🥸HTML의 <태그>란? HTML을 기술하기 위해서 사용하는 명령어의 집합을 <태그>라고 한다. <태그>의 종류는 다양하니, 하나하나씩 알아봅니다 head태그: 메타데이터를 포함하기 위한 태그, 웹페이지에 존재하며 화면에 표시되지 않는다. head태그에 주로 오는 요소들은 이러하다 **:**
  • title : html문서의 제목
  • style : html문서의 style정보
  • link : 외부 리소스와 연결 정보 (주로 css파일 연계시 사용)
    • ~~아직 사용해본 내용만 적어봤다.~~
body태그: 웹페이지를 구성하는 대부분의 요소를 body태그 내에 작성할 수 있습니다.
### ■ HTML 기본 문법📜: 한 태그의 **_속성_** 작성법은 이러합니다 **:** <태그 속성="값">내용 모든 태그들과 요소들을 작성할 때는 보통 <태그>로 작성하고 그 사이에 내용을 작성하는데, 그렇지 않은 요소를 **_빈요소_** 라고 합니다. **_빈요소_** 는 주로 br,hr,img,input,link등을 사용합니다. (안전성을 위해 빈요소 작성시에는 꼭 /을 요소 뒤에 넣는 습관을 기릅시다.ex: <태그/>)
### ■ HTML 태그 종류🤩: **기본 태그:**
  • h1,h2,h3,h4,h5,h6 : 제목 태그, 숫자 순서에 따라 글씨 크기가 달라집니다. (자동 줄바꿈 적용됨)
  • p : 본문 태그 (자동 줄바꿈 적용됨)
  • br : 줄 바꿈 태그(html문서에서 enter로 줄을 바꿀 수 없음)
  • ul : 순서 없는 목록, type속성을 통해 모양 변경가능. ex : type:"square, circle..."등 택 1
  • ol : 순서 있는 목록, type 속성 적용해 말머리 기호 변경가능.
    • 속성 사용 가능 (ol type: "1, A, a, I, i 중 택 1")
    • ol start="?" --- 시작 값 변경
    • ol reversed --- 역순으로 시작
  • li : ul이나ol사용시 리스트 항목을 만들어 주는 코드
  • hr : 수평 줄 만드는 태그

  • summary와details : 내용을 숨겨주는 태그
    • 이렇게 말이죠(클릭)👇 짜쟌~~~☆*: .｡. o(≧▽≦)o .｡.:*☆


  • 문자 꾸미기 태그
    1. b,strong : 글자를 두껍게! ;
    2. i,em : 글자를 기울게 ;
    3. del : 중간 줄 긋기;
    4. u : 밑줄 긋기;
  • img : 이미지를 넣을 때 사용하는 태그
    • src : src 속성은 img 파일의 인터넷 주소 링크나 파일위치를 작성하는 곳
    • alt : alt 속성은 정보를 더 주기 위한 속성
    src
    src 속성은 img 파일의 인터넷 주소 링크나 파일위치를 작성하는 곳
    alt
    alt 속성은 정보를 더 주기 위한 속성
  • a : 하이퍼링크 태그
    • href : 인터넷 주소 링크나 파일위치를 작성하는 곳
    • target : 새 탭 열기 여부 설정 ex. _blank,_self

🥺HTML form 태그 정리

  1. 입력값 input type👇
    • button : button의 type은 value로 이름 설정 가능.

      ex.

    • text : text는 placeholder로 내용 설정가능,문구와 칸이랑 연동 필요시, label을 사용해서 진행.

    • email : email은 이메일 작성 칸을 만드는 것, placeholder를 통해 예시문구 작성 가능.

    • password : password는 암호 작성 칸 만들기.

    • checkbox : checkbox는 여러 선택사항 선택 가능.

    • radio : radio는 하나만 선택가능.


🥹table태그 정리📊

  • tr : 표 내부의 행
    • table속성
    • border : 테두리 두께. ex : border: 1px solid black;
    • cellspacing : 테두리 간격 사이의 너비
    • cellpadding : 셀 내부의 간격
    • align : 테이블 정렬 속성
    • width,height : 테이블 너비와 높이
    • background-color와bordercolor : 테이블 배경색과 테두리 색
  • th : 표 내부의 제목 칸
  • td : 행 내부의 일반 칸
    • td속성
    • colspan : 해당 칸이 점유하는 열의 수를 지정함
    • rowspan : 해당 칸이 점유하는 행의 수를 지정함
    • 예시1
      td td td
      td td td
    • 예시2 :
      1 2
      3 center
      4
