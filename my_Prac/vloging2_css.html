W3C의 표준이고, 레이아웃과 스타일을 정의할 때의 자유도가 높다. 기본 파일명은 style.css이다. (출처:wikipedia) 간단하게 말하면 HTML문서를 꾸며주는 스타일 언어가 CSS다.
### 🏫CSS 사용방법 CSS를 사용할려면 3가지 방식이 있다. 인라인방식, 내장 스타일과 링크방식. html문서 내에서 태그안에 작성하는 _인라인 방식_, head태그 안에 작성하는 _내장 스타일_, head태그안에 link태그를 이용해 작성하는 _링크방식_ 이 있습니다. 그 외에도 @import를 통해 css문서 내에서 여러 css문서를 불러올 수 있다.

### 🤩CSS 선택자 선택자는 종류가 많고 작성할 때 많이 까먹어서 이번기회에 다시 복습 해볼려고 한다ㅠㅠㅠ **기본 선택자:**
  • * : 전체 선택자
  • AAA : 태그 선택자
  • .AAA : 클래스 선택자
  • #AAA : 아이디 선택자
**복합 선택자:**
  • AAABBB : 일치 선택자
  • AAA > BBB : 자식 요소 선택자, AAA의 자식요소 BBB를 선택하는 것
  • AAA BBB : 하위요소 선택자, AAA의 하위요소인 BBB 모두를 선택하는 것 (띄어쓰기 필수!)
  • AAA + BBB : 선택자의 다음 형제요소 중 하나 선택자
  • AAA ~ BBB : 선택자의 다음 형제요소 모두 선택자
**가상요소 선택자**
  • AAA:hover : 선택된 AAA에 마우스가 올라가 있는 동안 선택함.
  • AAA:active : 선택된 AAA를 마우스로 클릭하고 있는동안 선택함
  • AAA:focus : AAA가 포커스 되면 선택되는 선택자
  • AAA:first-child : 선택한 AAA가 형제요소 중 첫째라면 선택함
  • AAA:last-child : 선택한 AAA가 형제요소 중 막내라면 선택함
  • AAA:nth-child(n) : 선택한 AAA의 형제요소 중 (n)번째 선택하는 선택자 (AAA는 전체선택자*도 사용할 수 있음.)
  • AAA:not(BBB) : AAA요소 중 BBB가 아닌 요소 선택자
**속성 선택자**
  • [AAA] : 속성 AAA를 포함한 요소를 선택
  • [AAA="BBB"] : 속성 AAA면서 value가 BBB인 요소를 선택

### 🤯CSS 속성 CSS에 사용되는 속성은 너무 많아서 잘 모르거나 헷갈리는 것 위주로 다시 복습해 볼려고 한다.
  • line-height : 한 줄의 높이, 보통은 글과 글 사이에 (2줄이상) 간격을 정하는데 사용한다. 작성 방법은 아래와 같이 사용할 수 있다. ↓
``` line-height : 32px; line-height : 2; line-height : 200%; ``` **▼ display**👇 display는 CSS 배울때 제일 어렵다고 느낀거같다... display란, 요소가 회면에 출력되는 특성을 설정하는 건데..일단 display값으로 무엇이 있는지 알아보자. **↓↓**
  • block : 레이아웃 요소로 출력 됨
  • inline : 글자 요소로 출력됨
  • inline-block : 글자와 상자요소 동시에 출력함.
  • flex : 1차원 레이아웃, 따로 값을 지정하고 사용한다.
  • grid : 2차원 레이아웃, 위 flex와 동일하게 따로지정 필요.
  • none : 화면에서 사라짐 (visibility와 opacity랑 비슷하나 다르다.)
    • **▼ display : flex**👇 display : flex 를 사용할 때 지정하는 다른 요소들이 몇개 있는데, 이것도 역시 자주 까먹어서 다시 정리한다...
    • flex-direction : 축의 방향을 설정한다, row는 행 축,column은 열 축으로, 순서 번경을 원한다면 -reverse를 붙여야한다.
    • flex-wrap : 아이템의 줄바꿈 여부를 지정한다, nowrap은 줄바꿈 없음,wrap은 아이템의 크기를 지키면서 줄을 바꿔준다.
    • justify-content : 주 축의 정렬방법을 뜻하고, flex-start는 아이템을 시작점으로 정렬,flex-end는 끝 점에서 정렬, center는 아이템을 가운데로 정렬시킨다.이 외에도 space-between,space-around,space-evenly가 있다는 것을 기억하자.
    • align-items : 교차 축의 한 줄 정렬방법을 뜻한다, flex-start는 각 줄의 시작점으로 아이템 정렬, flex-end는 끝점으로 정렬, center는 각 줄의 가운데로 정렬한다.
    • align-content : 교차 축의 여러 줄 정렬방법, 다만 주의할 점은 flex-wrap이 wrap일 때 해당 값이 유효하다. 위 align-items와 같이 flex-start, flex-end, center,그 외에도 space-between,space-around가 있다.
